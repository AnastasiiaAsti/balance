# purpose:

A full stack CRUD application that utilizes a node/mongodb backend. Let's you add, update and store asanas

# ERD

A User has many Asanas; Asanas belongs to many users. <br>
Asana has many comments; Comments belong to one asana. <br>
User has many routines, routines belong to many users. <br>
Routines has many asanas; Asanas belong to many routines <br>

# Technologies Used

HTML, CSS, JS<br>
Bootstrap<br>
Node, Express<br>
MongoDB, Mongoose
