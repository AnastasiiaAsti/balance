# Mongoose-Yoga

link: https://mongoose-yoga.herokuapp.com/  </br>

A full stack CRUD application that utilizes a node/mongodb backend. Let's you add, update and store asanas. Currently redesigning UX/UI concepts. Utilizing mobile first approach

# ERD

A User has many Asanas; Asanas belongs to many users. <br>
Asana has many comments; Comments belong to one asana. <br>
User has many routines, routines belong to many users. <br>
Routines has many asanas; Asanas belong to many routines <br>
